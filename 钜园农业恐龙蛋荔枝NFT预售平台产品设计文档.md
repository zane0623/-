# 钜园农业恐龙蛋荔枝NFT预售平台产品设计文档

## 项目概述

### 项目背景
钜园农业是一家新加坡企业，在广东、广西、福建拥有大型农业生产基地。公司计划将特色产品"恐龙蛋荔枝"通过NFT/RWA（实物资产代币化）的方式进行预售，结合区块链技术实现透明化、可追溯的农产品预售服务。

### 产品定位
基于区块链技术的高端农产品NFT预售平台，连接钜园农业生产基地与全球消费者，提供优质恐龙蛋荔枝的预售服务。

### 产品名称
**钜园荔枝NFT** (Juyuan Lychee NFT)

### 产品Slogan
"新加坡品质，中国荔枝，全球预售"

## 合规框架设计

### 1. 多司法管辖区合规

#### 1.1 新加坡合规架构
**法律实体**：
- 钜园农业控股有限公司（新加坡）
- 数字资产发行许可证（MAS）
- 支付服务许可证（PSA）
- 反洗钱合规框架

**合规要求**：
- 符合新加坡《支付服务法》(PSA)
- 遵守《证券期货法》(SFA)
- 实施KYC/AML程序
- 定期合规报告

#### 1.2 中国合规策略
**生产基地合规**：
- 保持中国生产基地合法经营
- 确保农产品质量认证和溯源
- 遵守农产品出口相关法规
- 建立数据出境合规机制

**技术合规**：
- 使用海外服务器和区块链网络
- 实施数据本地化存储
- 建立数据安全传输机制

#### 1.3 跨境合规机制
**数据流合规路径**：
```
中国生产基地 → 数据加密 → 新加坡处理中心 → 区块链网络
                ↓
            合规审计报告 → MAS监管
```

### 2. KYC/AML详细流程

#### 2.1 用户身份验证
**验证标准**：
- **个人投资者**：年收入≥50万新币或净资产≥100万新币
- **机构投资者**：注册资本≥500万新币
- **风险等级**：低、中、高三级分类

**验证流程**：
1. 基础信息收集
2. 身份证明验证
3. 地址证明验证
4. 资金来源调查
5. 风险评估
6. 合规审查
7. 账户激活

#### 2.2 AML监控系统
**监控规则**：
- 大额现金交易监控
- 频繁小额交易检测
- 异常时间交易分析
- 高风险地区交易标记
