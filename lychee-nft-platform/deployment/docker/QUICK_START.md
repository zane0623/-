# 🚀 快速开始指南

> 3 分钟启动完整的 Lychee NFT Platform

## 📋 前置要求

- ✅ Docker 20.10+
- ✅ Docker Compose 2.0+
- ✅ 8GB+ 可用内存

## 🎯 三步启动

### 步骤 1: 进入目录
```bash
cd deployment/docker
```

### 步骤 2: 运行设置脚本
```bash
./setup.sh
```

### 步骤 3: 访问服务
```
🌐 前端:    http://localhost
�� 监控:    http://localhost:3001
🗄️  数据库:  http://localhost:5050
```

## 🎮 常用命令

```bash
make status    # 查看状态
make logs      # 查看日志
make health    # 健康检查
make urls      # 显示地址
```
