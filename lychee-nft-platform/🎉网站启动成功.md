## 💡 开发提示

### 前端开发
```bash
cd frontend

# 开发服务器（已运行）
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm start

# 代码检查
npm run lint
```

### 添加新页面
```bash
# 创建新页面
mkdir -p src/app/新页面
echo 'export default function NewPage() { ... }' > src/app/新页面/page.tsx

# 自动可访问: http://localhost:3000/新页面
```

### 添加组件
```bash
mkdir -p src/components
# 创建可复用组件
```
