## 📝 使用说明

### 搜索功能
1. 在 Header 搜索框输入关键词
2. 查看搜索建议或按 Enter 搜索
3. 在搜索结果页使用筛选和排序

### 筛选和排序
1. 进入预售页面
2. 点击"更多筛选"按钮
3. 设置筛选条件
4. 选择排序方式

### 用户个人资料
1. 连接钱包后，点击 Header 的"个人中心"
2. 查看和编辑个人信息
3. 管理账户设置和安全设置

### 通知中心
1. 点击 Header 右上角通知图标
2. 查看未读通知
3. 标记已读或删除通知

### 收藏功能
1. 在预售页面点击心形图标收藏产品
2. 点击 Header 的心愿单图标查看收藏
3. 从心愿单直接加入购物车
